# Tenos

Browser-basiertes Echtzeit-Multiplayer-Physikspiel mit WebGPU, autoritativer Serverarchitektur und isomorphem TypeScript-Stack.

> **Status:** Entwurf v1.0 – Februar 2026
> **Vormals:** "Antigravity"

## Tech Stack

- **Rendering:** Babylon.js 8 (WebGPU + WebGL2 Fallback)
- **Physik:** Havok (WASM)
- **ECS:** BitECS
- **UI:** SolidJS
- **Netzwerk:** Colyseus
- **Runtime:** Bun
- **DB:** PostgreSQL + Drizzle ORM, Redis
- **Bundler:** Vite
